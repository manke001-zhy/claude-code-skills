## 💡 工作原理

1. **你的消息** → Bot转发
2. **GPT-4o分析** → 返回JSON（操作类型、文件名、内容）
3. **执行** → 创建/写入/读取文件
4. **返回结果** → 发回Telegram

## 📝 GPT-4o理解的格式

```
用户说: "ai助手.txt写入123456"

GPT-4o返回:
{
    "action": "write_file",
    "filename": "ai助手.txt",
    "content": "123456",
    "explanation": "向ai助手.txt写入内容123456"
}
```
