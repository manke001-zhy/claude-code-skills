# Telegram 双向控制

通过 Telegram Bot 远程控制 Claude Code 并实现双向交互。

## 主要功能

- 💬 通过 Telegram 与 Claude 对话
- 📁 文件传输（上传/下载）
- ⚡ 执行 shell 命令
- 🤖 AI 对话和代码生成
- 🔔 实时通知推送

## 快速使用

```
启动 Telegram Bot
发送消息到 Telegram
执行远程命令
```

## 配置

需要配置：
- Telegram Bot Token
- Chat ID
- (可选) OpenAI API Key

## 技术栈

- python-telegram-bot
- OpenAI API
