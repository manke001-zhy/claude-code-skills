# Humanizer-zh

## AI 写作去痕工具（中文版）

> 消除 AI 生成文本的痕迹，让文字更自然、更像人类写作

这是一个 Claude Code Skill，专门用于消除文本中 AI 生成的痕迹，使文字更自然、更像人类写作。

### 功能特点

- **词汇替换**: 将 AI 常用词汇替换为更自然的表达
- **句式变化**: 增加句子结构的多样性
- **风格优化**: 使文本更流畅、更自然
- **人类特征添加**: 引入人类写作的个人特征

### 安装方法

1. 将此文件夹复制到 Claude Code skills 目录：
   - Windows: `C:\Users\<你的用户名>\.claude\skills\Humanizer-zh`
   - macOS/Linux: `~/.claude/skills/Humanizer-zh`

2. 确保 Python 3.6+ 已安装

### 使用方法

在 Claude Code 中直接使用：

```
请用 Humanizer 处理以下文本：
[你的 AI 生成的文本]
```

或者指定处理强度：

```
请用 Humanizer（轻度处理）处理：
[文本]

请用 Humanizer（重度处理）处理：
[文本]
```

### 处理强度

- **Light (轻度)**: 保留更多原始内容，只做必要修改
- **Medium (中度)**: 平衡处理，推荐日常使用
- **Heavy (重度)**: 最大程度人类化，可能改变较多

### 示例

**输入（AI 生成）：**
```
首先，我们需要明确目标。其次，制定详细的计划。最后，执行并监控进度。
```

**输出（人类化）：**
```
我们需要先明确目标，接着制定详细的计划，然后执行并监控进度。
```

### 技术细节

- 纯 Python 实现，无外部依赖
- 支持批量文本处理
- 可自定义词汇替换规则
- 保留原始文本的核心含义
