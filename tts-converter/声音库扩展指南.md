## ✨ 扩展声音库(通过语速模拟)

### 🧒 小孩声音
```python
# 小男孩
{
    'voice': 'zh-CN-YunxiNeural',
    'rate': '+30%',  # 1.3倍速
    'description': '小男孩(快语速)'
}

# 小女孩
{
    'voice': 'zh-CN-XiaoxiaoNeural',
    'rate': '+30%',
    'description': '小女孩(快语速)'
}
```

## 🔧 自定义语速

如果你觉得小孩或老人的语速效果不理想,可以手动调整:

在 `voice_matcher.py` 的 `assign_voices` 方法中修改:
```python
if age == '小孩':
    rate = '+40%'  # 更快的小孩声音
elif age == '老人':
    rate = '-30%'  # 更慢的老人声音
```

## ❓ 常见问题

### Q: 为什么没有专门的小孩/老人声音?
A: 微软Edge TTS只提供了8个中文基础声音,没有专门的小孩或老人声音。我们通过调整语速来模拟这些声音效果。

### Q: 语速调整可以更精确吗?
A: Edge TTS支持的语速范围是 -50% 到 +100%,你可以根据需要调整。

### Q: 能否使用其他TTS服务?
A: 可以!你可以集成其他TTS服务(如Azure TTS、Google TTS、讯飞等),它们可能提供更多声音选择。

### Q: 如何让小孩声音更像小孩?
A: 除了加快语速,还可以:
1. 在脚本中用更简单的词汇
2. 使用更短的句子
3. 在对话前添加"(童声)"等提示

### Q: 能否添加音调(pitch)调整?
A: Edge TTS的Python库不支持pitch调整,但你可以使用Azure Speech Service来获得更多控制选项。
