# 技能创建器

快速创建 Claude Code 自定义技能的工具。

## 主要功能

- 🏗️ 生成技能项目结构
- 📝 自动创建 SKILL.md 文档
- 🛠️ 包含示例代码和脚本
- 📦 标准化目录结构

## 快速开始

```
创建一个新技能
使用 skill-creator 生成项目模板
```

## 输出结构

```
my-skill/
├── SKILL.md          # 技能文档
├── README.md         # 说明文档
├── scripts/          # 脚本目录
└── package.json      # 配置文件
```

## 技术栈

- Python / Node.js
- 模板生成器
